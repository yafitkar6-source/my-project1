## 🎮 פרויקטים אינטראקטיביים

### 🤖 למידת מכונה (`machine_learning.html`)
ויזואליזציה אינטראקטיבית המסבירה את עקרונות למידת המכונה, כולל:
- סוגים שונים של אלגוריתמים
- תהליכי אימון
- ויזואליזציות של נתונים
- דוגמאות מעשיות

### 📈 מניות ישראליות (`israeli_stocks.html`)
מעקב ויזואלי אינטראקטיבי אחר מניות ישראליות, כולל:
- גרפים דינמיים
- ניתוח מגמות
- מחוונים לזמן אמת של שוק ההון
- נתונים מעודכנים

### 🧬 קיפול חלבונים (`protein_folding.html`)
ויזואליזציה תלת-ממדית אינטראקטיבית של תהליך קיפול חלבונים:
- בחינת המבנה המולקולרי
- סימולציה תלת-ממדית
- שליטה מלאה בתצוגה

### ♔ משחק שח מט (`chess.html`)
משחק שח מט אינטראקטיבי מלא עם:
- כל חוקי המשחק המלאים
- משחק שני שחקנים (לבן ושחור)
- זיהוי שח ושח מט
- ביטול מהלכים (Undo)
- הדגשת מהלכים אפשריים
- ממשק משתמש יפה ונוח

## 🔗 מאגר GitHub

הפרויקט זמין ב-GitHub:
**https://github.com/yafitkar6-source/my-project**

### העלאת קבצים ל-GitHub

יש מספר דרכים להעלות את הקבצים:

1. **העלאה ידנית דרך הדפדפן** (הכי קל)
   - לך למאגר ב-GitHub
   - לחץ על "uploading an existing file"
   - גרור את הקבצים

2. **שימוש ב-GitHub Desktop**
   - הורד מ: https://desktop.github.com/
   - פתח את התיקייה והעלה

3. **שימוש ב-Git**
   - הרץ את אחד מהסקריפטים: `.bat`, `.ps1`, או `.py`
   - או השתמש בפקודות Git ידניות

לפרטים נוספים, ראה את קבצי ההוראות בתיקייה.

# איך לפתוח את הפרויקט ב-GitHub

## קישור ישיר לפרויקט:
**https://github.com/yafitkar6-source/my-project**

---

## דרכים לפתוח את הפרויקט:

### שיטה 1: פתיחה ישירה בדפדפן
1. פתח את הדפדפן שלך (Chrome, Edge, Firefox וכו')
2. העתק והדבק את הקישור הבא בשורת הכתובת:
   ```
   https://github.com/yafitkar6-source/my-project
   ```
3. לחץ Enter

---

### שיטה 2: דרך GitHub
1. לך לאתר: https://github.com
2. התחבר לחשבון שלך (אם עדיין לא התחברת)
3. לחץ על האייקון של הפרופיל שלך בפינה הימנית העליונה
4. בחר "Your repositories" או "המאגרים שלך"
5. לחץ על "my-project"

---

### שיטה 3: חיפוש ב-GitHub
1. לך ל: https://github.com
2. לחץ על שדה החיפוש בראש הדף
3. הקלד: `yafitkar6-source/my-project`
4. לחץ Enter
5. בחר את המאגר מהתוצאות

---

## מה תראה כשתפתח את הפרויקט:

לאחר שתפתח את הפרויקט, תראה:

### 📁 קבצים בפרויקט:
- `index.html` - דף הבית עם תפריט הפרויקטים
- `machine_learning.html` - פרויקט למידת מכונה
- `israeli_stocks.html` - פרויקט מניות ישראליות
- `protein_folding.html` - פרויקט קיפול חלבונים
- `README.md` - תיעוד הפרויקט
- `.gitignore` - קובץ הגדרות Git

### 🔍 אפשרויות נוספות:
- **Code** - צפייה בקוד
- **Issues** - פתיחת בעיות/בקשות
- **Pull requests** - בקשות למיזוג שינויים
- **Actions** - אוטומציה ובדיקות
- **Settings** - הגדרות המאגר

---

## צפייה בקוד:

1. לחץ על שם קובץ כדי לראות את התוכן שלו
2. לחץ על "Raw" כדי לראות את הקוד הגולמי
3. לחץ על "Blame" כדי לראות מי כתב כל שורה
4. לחץ על "History" כדי לראות את ההיסטוריה של הקובץ

---

## עריכה ישירה ב-GitHub:

1. לחץ על קובץ כלשהו
2. לחץ על האייקון של העיפרון (✏️) בפינה הימנית העליונה
3. ערוך את הקוד
4. גלול למטה ולחץ על "Commit changes"
5. כתוב הודעה לתיאור השינוי
6. לחץ על "Commit changes"

---

## הורדת הפרויקט למחשב:

### דרך 1: Clone
1. לחץ על הכפתור הירוק "Code" בפינה הימנית העליונה
2. בחר "HTTPS" או "SSH"
3. העתק את הקישור
4. פתח Terminal/PowerShell והרץ:
   ```
   git clone https://github.com/yafitkar6-source/my-project.git
   ```

### דרך 2: Download ZIP
1. לחץ על הכפתור הירוק "Code"
2. לחץ על "Download ZIP"
3. חלץ את הקובץ למחשב שלך
